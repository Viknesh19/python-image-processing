# Image Processing GUI using Python

A simple image processing desktop application built using Python, Tkinter, and OpenCV.  
It allows users to open images (from file or URL), apply denoise and sharpen effects, preview changes, and save the final image easily.

---

## Features
- Open image from local file or URL  
- Apply denoise and sharpen filters using sliders  
- Revert to original image anytime  
- Save processed image in PNG or JPG format  
- Simple and clean user interface  

---

## Technologies Used
- Python 3  
- Tkinter  
- OpenCV  
- Pillow (PIL)  
- NumPy
